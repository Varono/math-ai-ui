# math-ai-ui
Frontend for Math AI Helper

## Τοπικά

**Εύκολο:** Στο φάκελο **Project** (πάνω από math-ai-ui / math-ai-server) τρέξε **`run-all.bat`**. Ξεκινάει API + UI και ανοίγει το browser.

**Χωριστά:**
1. **API:** `cd math-ai-server` → `start-server.bat`.
2. **UI:** `cd math-ai-ui` → `serve-ui.bat`. Άνοιξε **http://localhost:8080**.

Μη ανοίγεις το `index.html` ως αρχείο (`file://`). "Failed to fetch" = API δεν τρέχει ή UI από file://.
